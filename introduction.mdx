## What is Typeset?

Typeset is a modern LaTeX editor that runs entirely in your browser. Write, compile, and collaborate on LaTeX documents without any setup or downloads. Built with Next.js and powered by cutting-edge AI, Typeset brings real-time collaboration and intelligent assistance to academic and technical writing.

Experience LaTeX editing reimagined for the modern web at [typeset.im](https://typeset.im).

## Tech stack

Typeset is built with modern web technologies:

- **Next.js** - React framework for the web application
- **TypeScript** - Type-safe development
- **Clerk** - User authentication and management
- **Liveblocks** - Real-time collaboration infrastructure
- **AI SDK** - Unified interface for AI models
- **CodeMirror** - Advanced code editing
- **Tectonic** - Fast LaTeX compilation
- **shadcn/ui** - Beautiful UI components
